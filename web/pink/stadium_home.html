인천 계양체육관(배드민턴체육관)

흥국생명 핑크스파이더스의 연고지는 인천이며,천 일원 계양체육관(배드민턴체육관)을 홈경기장으로 이용하고 있습니다.

  • 주소 : 인천광역시 계양구 서운동 110번지 일원 계양체육관(배드민턴체육관)

  • 일반버스 :
    계산역 2번 출구 / 작전역 4번 출구 → 버스환승 (88) → 서운체육관 하차
    계산역 2번 출구 / 작전역 4번 출구 → 버스환승 (588) → 영음교회 하차
    송내역2번출구(100미터 전방) → 버스환승 (302) → 서운동 체육관 하차

  • 셔틀버스 :
    작전역 3번 출구(경기시작 2시간 30분 전 부터)
